# AGENTS.md

Stack: Vite + React 19 + TanStack Router + TanStack Query + Shadcn UI (new-york style) + Tailwind v4 (CSS-based config) + Zod + react-hook-form + Zustand. `package.json` name: `eburon-ai`. TS ~6.0.

## Commands

Package manager is **pnpm**. Ignore `package-lock.json`, never `npm install`.

```bash
pnpm install
pnpm dev                        # Vite dev server
pnpm build                      # tsc -b (typecheck) THEN vite build
pnpm lint                       # eslint .
pnpm format                     # prettier --write .
pnpm format:check               # prettier --check .   (CI uses this)
pnpm knip                       # unused files/deps
pnpm test:browser:install       # playwright install chromium --with-deps
pnpm test <path>                # vitest run --browser.headless
pnpm test:ui                    # vitest --ui --browser.headless
pnpm test:coverage              # vitest run --coverage --browser.headless
```

CI order: `lint → format:check → test:browser:install → test → build` (`.github/workflows/ci.yml`). knip step commented out.

## Tests

- Browser-based (chromium via Playwright, `@vitest/browser-playwright`), NOT jsdom.
- `render` from `vitest-browser-react`, `userEvent` from `vitest/browser`. Do **not** import from `@testing-library/react` or `@testing-library/user-event`.
- Assertions: `await expect.element(...)` (vitest-browser API).
- Colocated: `*.test.tsx` for components, `*.test.ts` for libs. Shared helpers in `src/test-utils/`.
- Coverage excludes: `src/components/ui/**`, `src/routes/**`, `src/routeTree.gen.ts`, `src/test-utils/**`, `src/assets/**`.

## Generated / vendored — do not hand-edit

- `src/routeTree.gen.ts` — generated by `@tanstack/router-plugin/vite` (`autoCodeSplitting: true`). Has `@ts-nocheck` + `/* eslint-disable */`.
- `src/components/ui/**` — Shadcn UI. Do **not** blind-overwrite with `npx shadcn add`. Modified: `scroll-area`, `sonner`, `separator`. RTL-updated: `alert-dialog`, `calendar`, `command`, `dialog`, `dropdown-menu`, `select`, `table`, `sheet`, `sidebar`, `switch`. All others standard.
- `.tanstack/tmp/` — router plugin temp dir.

## Architecture

**Entry (`src/main.tsx`):** Provider stack: `ClerkProvider → QueryClientProvider → ThemeProvider → FontProvider → DirectionProvider → RouterProvider`. Router built from `routeTree.gen.ts`, context `{ queryClient }`. 500 errors navigate to `/500` in prod.

**Routes (`src/routes/`, file-based):** `__root.tsx` is root. `_authenticated/` is auth layout. `(auth)`/`(errors)` are pathless route groups. `clerk/` is optional Clerk demo route.

**Features:** `src/features/<domain>/` — `apps`, `chats`, `dashboard`, `database`, `errors`, `issues`, `mail`, `models`, `settings`, `tasks`, `users`. Structure: `components/`, `data/`, `index.tsx`.

**Lib:** `src/lib/` — supabase client + queries, users-db, handle-server-error, cookies, utils, clerk-users-api, hostinger-mail. Tinybird analytics queries live in `src/lib/supabase-queries.ts`.

**API (`api/`):** Vercel serverless function catch-all routes (`api/clerk/[...path].ts`, `api/mail/[...path].ts`). In dev, served by Vite proxy middleware: `/api/mail` → `api.mail.hostinger.com`, `/api/clerk` → `api.clerk.com/v1`.

**Alias:** `@/*` → `src/*`.

## Prettier / ESLint

- **No semicolons**, single quotes (incl. JSX), 2-space, printWidth 80, trailingComma `es5`, LF.
- **Import order** (`@trivago/prettier-plugin-sort-imports`, enforced by `pnpm format`): `react` → `@vitejs/*` → `globals` → `zod` → `axios` → `date-fns` → `react-hook-form` → `@radix-ui/*` → `@hookform/resolvers/zod` → `@tanstack/react-query` → `@tanstack/react-router` → `@tanstack/react-table` → `THIRD_PARTY_MODULES` → `@/assets` → `@/api` → `@/stores` → `@/lib` → `@/utils` → `@/constants` → `@/context` → `@/hooks` → `@/components/layouts` → `@/components/ui` → `@/components/errors` → `@/components` → `@/features` → relative.
- Tailwind classes sorted via `prettier-plugin-tailwindcss` (functions `cn`, `clsx`).
- ESLint: `no-console: error` (intentional uses need `// eslint-disable-next-line no-console`). Type imports: `import type { X }` or `import { type X }`. Unused vars: prefix with `_`. `no-duplicate-imports: error`. `src/components/ui/` excluded from ESLint.

## Env

Copy `.env.example` → `.env`:

- `VITE_CLERK_PUBLISHABLE_KEY` — optional. App runs without it (shows setup at `/clerk`). Lives in `src/routes/clerk/` + provider in `src/main.tsx`. Fully removable.
- `VITE_SUPABASE_URL`, `VITE_SUPABASE_ANON_KEY` — client in `src/lib/supabase.ts`, also accepts `NEXT_PUBLIC_*` fallbacks. DB schema in `supabase/schema.sql`.
- `CLERK_SECRET_KEY` — used by dev API proxy (`/api/clerk`) and Vercel function (`api/clerk/[...path].ts`).
- `HOSTINGER_MAIL_TOKEN` — used by dev API proxy (`/api/mail`) and Vercel function (`api/mail/[...path].ts`).

## Deploy

- Both platforms deploy `pnpm build` → `dist/`.
- Netlify: `netlify.toml` redirects `/*` → `/index.html`.
- Vercel: `vercel.json` rewrites `/api/*` → `/api/$1`, everything else → `/index.html`.
